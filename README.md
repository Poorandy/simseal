# simseal
> 数值模拟平台

 - [x] 前端返回模拟需要的卡牌列表、角色信息、怪物列表、模拟次数
 - [x] 单位类
 - [x] 卡牌类
 - [x] 事件执行器
 - [ ] 每回合费用没有用完
 - [ ] 卡牌墓地
 - [ ] 目标类型:敌对（友方）非死亡目标、敌对（友方）全部目标、自己
 - [x] 事件优先级
 - [x] 战斗流
